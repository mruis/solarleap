Next steps

Take a moment to read the below guidance to get the most out of your new solar power.

  1. Think which appliances you could operate during daytime - there is the most solar power available then. For example, your dishwasher or washing machine. The SolarLeap application you are going to access next can help you to figure these appliances out.
  2. Add these appliances as your favorites so you can easily access them through the SolarLeap application when you want to run them!
  3. When you want to use an appliance (for example, your dishwasher) and want to run it next time there is solar power, go to the SolarLeap application and set it to run as soon as possible when there is solar power available - or set them to run at a later time.

The appliances you now run with solar power have no usage costs and they are totally emission-free. Congratulations once more!
